Real Dasshutsu Game x Nintendo 3DS: Chou Hakai Keikaku Kara no Dasshutsu (translation: Real Escape Game × Nintendo 3DS: Escape from the Plan of Mega Destruction) was produced as a free to start downloadable Nintendo 3DS software in Japan with association with SCRAP, associated with the "Real Escape Game" brand; and based on the escape room concept in which people have to solve puzzles in order to escape from a room.

It is a multiplayer game in which people team up to investigate a mystery about a "Program Bomb" that was installed in Nintendo 3DS systems, causing them to explode. More episodes can be purchased with real money, and the first episode was (for a limited time only) initially available for free.

Black screen glitch

A glitch where after moving to a new screen from a specific scene [clarification needed] in the game, the game would freeze on a black screen. This was fixed in the v1.2 patch. (source)

Please note that these glitches only exist in the Japanese versions of this non-Pokémon game, or are otherwise from a game which was only released in Japan.
