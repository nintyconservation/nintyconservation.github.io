Rival preset name 0x05 in Pokémon Red

Glitch preset names are inaccessible glitch names for the player and rival in the Pokémon games.

In Generation I

In Pokémon Red, Blue, and Yellow, by modifying the value of CC26 (cursor position), it is possible to select out of bounds preset names from the new game sequence, where 0x00 is the top option on the menu. While many of these names appear to be constant, some of the later ones may be sourced from RAM and may be random (see chaotic name).

These names are possible to see with a cheating device (such as with GameShark code 01xx26CC), or memory editor, or theoretically arbitrary code execution.
