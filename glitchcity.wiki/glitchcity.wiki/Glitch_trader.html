Glitch traders are invalid NPCs in the Pokémon series who will trade glitched Pokémon for a specific species (which is either a valid Pokémon or glitch Pokémon). The only known ways to access a glitch trader are by using arbitrary code execution or a cheating device.

Some glitch traders have a set of glitch dialog for requesting and receiving a Pokémon. Others use one of the existing in-game trade messages.
