In Pokémon Yellow

Offgao's memory editor, also known as Pocket Computer is a memory editor GUI for arbitrary code execution in the Generation I Pokémon games.

It was originally created by offgao for Pokémon Red and Green, but was ported to non-Japanese version Pokémon Red, Blue, and Yellow by Cryo and Torchickens (ChickasaurusGL).

How to use it

  • Scroll down/up one address with the d-pad
  • Scroll 0x10 addresses with left and right
  • Hold Start and use up/down to increase or decrease the address by one (changes are applied in real time). Left/right allows you to increase or decrease the address by 0x10.
  • Hold Select and move up/down to swap the value of one address with the other adjacent address.

Glitches in editor

  • Offgao's memory editor itself has a glitch in it. If the Town Map is viewed directly before using the memory editor, the characters in the name that aren't letters may be replaced with Town Map tiles. This is similar to how offgao's memory editor isn't fully supported for in battle. While it is usable, it may again use the wrong tiles.
