Trainer escape Trainer Pokémon finder

From Glitch City Wiki
Jump to navigationJump to search

The Trainer escape Trainer Pokémon finder is an interactive tool that was created by Isaac356 for use with the Special stat encounter part of the Trainer escape glitch.

It was intended to be an alternative to wwwxxyy and Ryumaster's Trainer escape glitch yield image and uses Trainer data from the Pokémon Red disassembly project.

This tool provides a drop-down box where the user can select a Pokémon. After choosing a Pokémon, the tool will create a list of valid Trainers and their Pokémon which will cause the glitch to result in the Pokémon chosen from the drop-down box if it was the last Pokémon encountered.

It is currently hosted on the website Exocron.me.
