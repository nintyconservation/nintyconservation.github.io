MetascriptDex/RB:D653 (Lance's Pokémon League room)

(↑ Back to the MetascriptDex index)

This article details various glitch meta-map scripts for Lance's Pokémon League room in Pokémon Red and Blue.

The script is controlled by memory address D653.

Highlights

  • 0x01: Trainer escape glitch compatible script
  • 0x04, 0x0C: Unknown/can walk in room freely
  • 0x05: Arbitrary code execution at FAC9 (DAC9)
  • 0x06: Arbitrary code execution at D866
  • 0x07: Game may freeze
  • 0x08: Arbitrary code execution at C7A0
  • 0x09: Game may freeze
  • 0x0A: Arbitrary code execution at BFCD
  • 0x0B: Arbitrary code execution at D234
  • 0x0D: Arbitrary code execution at E0AF (C0AF)
  • 0x0E: Arbitrary code execution at FAB4 (DAB4)
  • 0x0F: Arbitrary code execution at CD3D
  • 0x10: Arbitrary code execution at AFCD
  • 0x11: After a short delay, arbitrary code execution at B4E0
  • 0x12: Game may freeze
  • 0x13: Game may freeze
  • 0x14: Turn changes to have a green/blue tint with white player sprite, a 'flicker line' effect can be seen (like the lines when trying to take a photo of a television)
  • 0x15: Execution of FE29 in OAM
