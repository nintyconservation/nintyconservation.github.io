The AttackDex is a database of glitch moves in the Pokémon games.

Effect bytes in this table are marked with special tags. The meanings are as such:

  • D: damaging move
  • S: status move
  • NE: no effect
  • ACE: capable of arbitrary code execution
  • F: always freezes the game (if move does not knock out the opponent)
  • PE: partial escape glitch move
  • CS:XX: changes a glitch stat, XX represents the hex ID of the glitch stat.
  • RS: requires a sleeping foe to hit
  • CT: has a charge turn
  • RRC: requires recharge
  • SD: halves foe's Defense and causes user to faint
  • SFE: halves foe's HP or deal 1 damage if already at 1 HP (Super Fang's effect)
  • PDE: deals damage and gives user's level*2*number of PDE (Pay Day effect) move uses money after battle

Please note: This project was started before the site transitioned into a wiki but is not complete.

While all articles for Pokémon Red, Blue, and Yellow are complete, the table is yet to be completed. The older (now obsolete) listing can be found here (link).
