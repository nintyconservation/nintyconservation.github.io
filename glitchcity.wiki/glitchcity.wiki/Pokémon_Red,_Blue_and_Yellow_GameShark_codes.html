This list of GameShark codes for Pokémon Red, Blue, and Yellow based on Matthew Robinson's GameShark Code Archive (text file).

Other codes have been taken from elsewhere.

Many Red and Blue codes (except things such as the walk through walls code 010138CD, which is the same in Yellow) can be used in Yellow by subtracting 1 hex digit from the 6th digit in the code.

Likewise, many Red, Blue and Yellow codes work on non-English language versions, if 5 is added to the 6th digit, when considering that the values are in hexadecimal. Example: 01??5ED3 is 01??63D3 in German Red/Blue. The equivalent code for German Yellow is 01??62D3.

'??' or 'xx' should be replaced with a value (see the Big List for Pokémon, move, item and letter values).

To find a code in this huge list, use your browser's search feature. Ctrl+F may bring it up.

All of the codes on this list start with 01. This beginning should work on most games, but if it doesn't work, change it to 91. One of the two should work.

See also Data Crystal's Pokémon Red and Blue RAM map for memory addresses on their own.

Please report any broken codes on this article's talk page or on the forums.
