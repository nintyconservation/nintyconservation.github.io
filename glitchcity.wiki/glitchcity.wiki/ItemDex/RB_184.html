(hex:B8) is a glitch item in Pokémon Red and Blue. Its name features a different glitch tile in the second character of its name to the otherwise identically named ItemDex/RB:183.

When this glitch item is used outside of battle, it will act as the same as a Super Repel (a Repel for 200 steps), and one of this glitch item will be used up. Attempting to use it from within battle will instead result in Professor Oak's message that it isn't the time to use the item.
