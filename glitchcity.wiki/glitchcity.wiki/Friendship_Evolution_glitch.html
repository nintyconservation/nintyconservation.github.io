The Friendship Evolution glitch is a glitch in Generation VI core games.

Pokémon transferred from one game to another using Pokémon Bank will not be able to evolve by friendship, if the save game on the other game has a different OT.

This does not apply to Pokémon obtained with Poké Transporter.
