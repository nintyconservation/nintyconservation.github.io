MetascriptDex/RB:D667 (Route 23)

From Glitch City Wiki
Jump to navigationJump to search

(↑ Back to the MetascriptDex index)

This article details various glitch meta-map scripts for Route 23 (near Pokémon League) in Pokémon Red and Blue. The script is controlled by memory address D667.

  • 0x00-0x02: Unknown (no known effect)
  • 0x03: Walking lag with glitch sounds
  • 0x04: Arbitrary code execution at FA52 in Echo RAM (DA52)
  • 0x05: ACE (D361 in WRAM)
  • 0x06: Unknown (no known effect)
  • 0x07: Invalid opcode at 0E04. If on Virtual Console game might not freeze.
  • 0x08: Game freeze
  • 0x09: ACE (8000 in VRAM)
  • 0x0A: ACE (CDC0)
  • 0x0B: ACE (B80D in SRAM)
  • 0x0C: ACE (F720 in Echo RAM (D720))
  • 0x0D: Unknown (no known effect)
  • 0x0E: Slight delay until ACE at C9D6
  • 0x0F: Invalid opcode at 14:62FC. If on Virtual Console, game may freeze in a different way
  • 0x10: Game attempts to run FED3 in inaccessible RAM
  • 0x11: ACE (D00E)
  • 0x12: ACE (E07B in Echo RAM (C07B))
  • 0x13: ACE (8000)
  • 0x14: Unknown (no known effect)
  • 0x15: Similar or the same as 0x0E
