This article details various glitch meta-map scripts for Route 18's gate in Pokémon Red and Blue. The script is controlled by memory address D669.

  • 0x00: (Possibly if no Bicycle, and locked): "Excuse me!" text
  • 0x01: "You need a BICYCLE for CYCLING Road!", move player step one step right unless impassable tile like a wall
  • 0x02: Similar to or the same as 0x01
  • 0x03, 0x04, 0x08, 0x12: Can proceed with no message, even if no Bicycle
  • 0x05: Arbitrary code execution (C057)
  • 0x06: ACE (CC21)
  • 0x07: ACE (CD58)
  • 0x09: Game freeze
  • 0x0A: ACE (Echo RAM: E002 (C002))
  • 0x0B: ACE (CD8C)
  • 0x0C: Glitch dialog that eventually ends with the normal message that "No pedestrians are allowed on CYCLING ROAD!"
  • 0x0D: ACE (E0AF (C0AF))
  • 0x0E: ACE (FAB4 (DAB4))
  • 0x0F: ACE (CD3D)
  • 0x10: Game may reset itself
  • 0x11: ACE (VRAM: 8000)
  • 0x13: Invalid opcode at 3DD0 (freeze). Unknown behavior on Virtual Console
  • 0x14: Invalid opcode at 38EB (freeze). On Virtual Console may be a different type of freeze
  • 0x15: Input disabled for a while, until game freeze
